.. rst-class:: lead

Collection of components to build graphical user interfaces (GUIs) in Rhino 8 for COMPAS packages.

The package provides basic forms for the visualisation of and interaction with
data stored in COMPAS geometry objects and data structures.

It also provides extensions of standard COMPAS scene objects that
link COMPAS geometry objects and data structures with their representations in Rhino,
and keep corresponding objects synchronised durinh user interaction.
